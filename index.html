开设模具通用步骤及注意事项

发表于 2026-02-07 分类于 感悟 Valine：

  • 总方向：方便模具工组装和产品制作及脱模

  • 根据深化图纸，初步设计开模方案，选择整体 或 分件制作。

    深化图设计另详。仔细查看深化图中的尺寸：长、宽、高、厚、弧度、转角、凹凸、数量、预埋件、吊件、分布筋、周围环境（地上、地下、埋深、排水）等状况，初步确定选择地模 或 平台模，综合考虑适合的开模方案。

    • 简易法：根据产品通长方向设置定位板、分布筋板并附加面板，适用于规则简单、单一外凸类（或凹凸不大）的产品。
    • 叠加法：由“简易法”延申而来，在产品局部拐角、突变处，因其特殊形状而增加设置定位板、筋板并附加面板。需要注意的是，有时仍然会遗漏，比如整体产品为长条形的，很自然的就会漏掉“长纵向”加筋板的可能性。谨记：有弧度必有底（筋板）。
    • 分模法：亦称“活模法”，针对多组产品、“瘦高”形产品，采用分块模具组合成整体方式制作模具，方便对犄角旮旯地方抹灰整平、喷涂，且可重复利用。
    • 内嵌法：针对凹形产品，当采用其他方法制作模具过于麻烦时，可采用此法。特点是模具外侧平整，仅需内部嵌入相应尺寸的垫板，达到所需的“凹形效果”即可。模具制作时亦方便省事。

  • 在与模具工协商确认好方向后，启动开模设计

    先将产品倒置（在CAD中则先将其镜像），然后按照定位板（底板）、筋板（支撑板）、分模板（如有）、面板（产品接触板）、盖板（上定位板）、分隔板（挡板）、封头板（堵头）、刮板（一般为弧形）顺序设计。

    • 定位板：作为模具中的基础，定位板定位尺寸很重要，一步错步步错。一般用15mm板雕刻，设15cm宽，定位口15mm宽，一般为封闭的整体。部分平台模设计时会考虑“卡口设计”，届时需预留1~2mm间隙，以防插不下去，且定位板根据活模特性为分段式。定位口的设置需综合考虑筋板与面板，防止打架。分割及组装参照前篇所述。

    • 筋板：作为模具中的支撑骨干，且是面板的基础，也很重要。设计时需考虑让出面板、盖板的厚度，犀牛设计时需采用“偏移面”的方法。一般会在产品的外侧，将筋板侧边开豁口，方便模具工组装。一般用15mm板雕刻。需注意产品的突变处需加设筋板。

    • 分模板：根据产品数量、特点选择合适厚度的分模板，一般分为15mm、18mm厚。位置一般从方便分模的地方下移1~3cm断开分模，长度上跟随产品长度，宽度上则根据具体情况设置通长或局部宽度。需注意分模的上下两部分需方便对接，且接头点的缝隙有保证。对于部分过深的产品，是否多次分模可以与模具工商量。

    • 面板：面板的分割需考虑产品成型时的缝隙、底部筋板的支撑，尽量少分割且分割处下方有筋板支撑。各个面板的搭接顺序需考虑清楚并于断面图上显现。

    • 盖板及封头板厚度根据需要调整，封头要区分封闭头还是分段头，二者挡板不一样。分段封头是否考虑刚性做成封闭的结合实际情况处置。

    • 所有模具的分割处理是否垂直切割，根据具体产品而定。一般情况下垂直切割。

  • 复核产品模具图形及尺寸

    按照顺序回想一遍，参照3D模型及CAD图纸，并互相印证，确保每一步，尤其是节点部位准确无误。尺寸是否正确、是否有遗漏、相互间是否打架等。

  • 输出雕刻及出图

    • 依然按照定位板（底板）、筋板（支撑板）、分模板（如有）、面板（产品接触板）、盖板（上定位板）、分隔板（挡板）、封头板（堵头）、刮板（一般为弧形）顺序输出外框或封闭多段线，分类到5、9、15、18mm等规格并编号，分类2D排版、CAD布局并出图、AI导出、JD程序、板材雕刻。过程中亦需要检查有无错误，比如排版镜像、缺棱掉角、导出尺寸、雕刻异常等。
    • 整体三维效果图打印，包括模具3D和产品3D图；对应CAD详图打印；底板标注三角尺寸并打印，方便模具工定形；筋板断面图标注各面板编号并打印，需详细指出各个面板的搭接关系、筋板的内外关系；面板、盖板、分隔板的整体布局关系图。
    • 时常到模具工处看看进展情况并询问是否存在问题，及时发现根源及时解决。

模具定位板

发表于 2026-01-14 分类于 模具 Valine：

异形预制构件采用模具制作，模具安装师傅根据地面定位板，将筋板、面板、固定板等按位安装。

定位板的分割雕刻需考虑组装时的偏差问题，一般拐角处、变截面处及其他复杂地方需要整体雕刻，不能切断，否则会导致误差偏大。例如木模具切割时的毛边，本身的柔性等都会影响模具最终的尺寸。

定位木板的组装根据图纸尺寸，及时微调，保证最终长度、宽度及水平度符合图纸要求。

组装完毕需复核尺寸。

标高引起的连锁反应

发表于 2026-01-10 分类于 感悟 Valine：
  • 问：为什么设计图纸一般都会开门见山的写设计说明？
  • 答：介绍下工程概况、参照规范、一般做法、其他在详图中不能简洁表达的说明。

那么好像只要在图中表达清楚要做什么、怎么做，设计说明也不太需要了。

今天发生的事，让我觉得设计说明这东西存在即合理-冗余说明好歹多一份保障。

一早对方就通知到场的板材尺寸错误，不符合原图标注的“相对标高2.60/2.80/3.00”，质问怎么下料的。闻言我先是一懵，这都2026年过一周了，合着需要你确认的深化图你根本没看么？

上图这么清楚的标注都无视吗？那当时确认什么的？

退一步讲，来看看原图，平面图确实标注了“相对标高2.60/2.80/3.00”，但也同时标注了“2600/2800/3000高×1000×140”，结合下方断面图地面以上标注“2249”，由此理解成板材实际高度就是2600/2800/3000高不合理吗？

回到平面图的“相对标高2.60/2.80/3.00”，此为对方纠结点：“相对标高必然是和地面相对的高度。”我表示理解，但严格来说，相对标高指的是和±0.000的高差，且原图并非采用标准标注符号和3位小数表示，而是汉字“相对标高”加上含2位小数的数字表示。一般来说，仍理解为从地面以上2.6/2.8/3.0。但是，图纸不是孤立的，需要再加立面和剖面综合来看。结合上图，理解为总高度情有可原。

那么对方为什么不反省自己不看图纸，反倒怪罪于我呢？

设计说明的好处就凸显了。它简洁还明了，放到够大，打开图纸就看到那种。若能把规格尺寸明确表示其中，此事当成笑谈了。
